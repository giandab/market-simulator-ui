# Market Simulator UI

The UI component of the Market Simulator App.

## Environment Setup

- Ensure the backend server is running: see https://github.com/giandab/Market-simulator .
- Run `npm i` to install dependencies.
- Run `npm run dev` to start the development server.

## Playwright Tests

- Ensure the development server is running
- Run `npx playwright test` or `npx playwright test --ui` to view the tests in the ui.
